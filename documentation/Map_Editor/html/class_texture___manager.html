Detailed Description

This class allows you to use textures in the program.

Version
1.0
Date
2, July, 2020
Warning
Many operation have asymptotics O (log n) as it uses a std::map.

This class has a enum with possibble names of textures. It also has a std::map which contains pairs ID-number - unique_ptr to the texture.
